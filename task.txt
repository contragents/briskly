Задача: сделать конвертацию файлов из out (формат 1С) в один json-файл (наш формат). Наш формат см. в папке in.

Дополнительные требования:
1. Использовать PHP 7+.
2. Zip-файлы нужно разархивировать внутри кода, а не вручную.
3. В обработку файлов добавить проверку контрольного числа штрих-кода (EAN13). Если есть ошибка, то логировать эту ошибку (и невалидный штрих-код).
4. Добавить в файл данные остатков по каждому товару (id склада + количество). Формат этого расширения придумать самостоятельно.
